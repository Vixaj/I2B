Windows Vista

Obsah

Co je to?

Windows Vista je operační systém od Microsoftu, který byl vydán v roce 2007.
Označení vista znamená výhled, rozhled.
Tento systém, byl určen pro počítače domácí, firemní či mediálních center.
Windows Vista byl vyznačován značnou nekompatibilitou se starším software i hardware a vysokými hardwarovými nároky, které zapříčinily nízký výkon (ve srovnání se starším systémem Windows XP).
Vista se nikdy nezbavila špatné pověsti u koncových uživatelů.

Kritika Visty

Vista, byla kritizována za její cenu, nekompabilitu s některým hardwarem i softwarem či vysokými hardwarovými požadavky.
Problémy byly způsobeny nekompabilitou se starším softwarovým vybavením, a také že jen málo lidí si vědělo rady se systémem, když se objevily potíže.
